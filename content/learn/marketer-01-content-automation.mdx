# 콘텐츠 제작, AI가 80%를 해줘

마케터의 가장 큰 시간 소비는 콘텐츠 제작이에요. 하나의 아이디어를 블로그, 인스타그램, 링크드인에 맞게 각각 다르게 써야 하죠. Claude Code는 하나의 핵심 메시지로 모든 채널용 콘텐츠를 한 번에 만들어줍니다.

<Callout type="tip">
AI가 초안을 만들고, 여러분이 브랜드 목소리에 맞게 다듬는 것이 가장 효율적이에요. 처음부터 완벽한 결과물을 기대하기보다 빠른 초안 → 빠른 수정 흐름을 만드세요.
</Callout>

---

## 1단계: 브랜드 보이스 파일 만들기

콘텐츠를 일관성 있게 만들려면 먼저 브랜드 가이드를 설정해요:

```
> 우리 브랜드 콘텐츠 가이드라인 파일을 만들어줘.
  
  brand-voice.md로 저장.
  
  우리 브랜드 정보:
  - 브랜드명: [이름]
  - 업종: [업종]
  - 타겟: [타겟 고객]
  - 톤: 친근하면서도 전문적 (예: "~해요" 체, 어렵지 않은 용어)
  - 절대 쓰지 않는 표현: [피해야 할 표현]
  - 우리다운 표현: [자주 쓰는 표현]
  - 핵심 메시지: [회사 슬로건 또는 핵심 가치]
```

---

## 2단계: 하나의 아이디어로 여러 채널 콘텐츠 생성

```
> brand-voice.md를 읽고
  다음 아이디어로 각 채널용 콘텐츠를 만들어줘.
  
  아이디어: [제품 신기능 출시 / 이벤트 / 인사이트 등]
  
  채널별 결과물:
  1. 블로그 포스트 (800-1200자, SEO 키워드 포함)
  2. 인스타그램 캡션 (150자 이내 + 해시태그 20개)
  3. 링크드인 포스트 (300-500자, 전문성 강조)
  4. 트위터/X 스레드 (5개 트윗 구성)
  5. 뉴스레터 섹션 (200자 요약)
  
  content-[날짜]-[주제].md로 저장.
```

---

## 3단계: 콘텐츠 캘린더 자동 생성

```
> 다음 달 콘텐츠 캘린더를 만들어줘.
  
  조건:
  - 주 3회 인스타그램 포스팅
  - 주 1회 블로그 포스팅
  - 주 2회 링크드인
  
  [업종/제품] 관련 주제로 4주치 콘텐츠 주제 추천.
  
  형식: calendar-[월].md (날짜, 채널, 주제, 키메시지 포함)
```

---

## 4단계: 콘텐츠 초안 일괄 생성

```
> calendar-[월].md의 콘텐츠 주제를 읽고
  이번 달 모든 콘텐츠 초안을 만들어줘.
  
  brand-voice.md 가이드라인 적용.
  각 콘텐츠를 content/[월]/ 폴더에 저장.
  파일명: [날짜]-[채널]-[주제].md
```

---

## 5단계: 성과 있는 콘텐츠 패턴 분석

```
> 지난 3개월 콘텐츠 성과 데이터를 분석해줘.
  
  파일: analytics-data.csv
  컬럼: 날짜, 채널, 제목, 도달수, 참여율, 클릭수
  
  분석 결과:
  1. 채널별 평균 참여율
  2. 가장 잘 된 콘텐츠 유형 Top 5
  3. 최적 포스팅 시간대
  4. 다음 달 전략 추천
  
  insights-[월].md로 저장.
```

<BeforeAfter>
  <Before>
    콘텐츠 1개 작성: 아이디어(30분) + 초안(2시간) + 수정(1시간) + 채널 변환(1시간) = 4.5시간
  </Before>
  <After>
    아이디어 정리(15분) + Claude Code 생성(5분) + 검토/수정(30분) = 50분
  </After>
</BeforeAfter>

## 실전 팁: 경쟁사 콘텐츠 분석

```
> Tavily로 [경쟁 브랜드명] 인스타그램/블로그의
  최근 콘텐츠를 검색해서 분석해줘.
  
  분석 포인트:
  - 어떤 주제가 반응이 좋은가
  - 어떤 형식(카드뉴스/릴스/포스트)을 많이 쓰는가
  - 어떤 해시태그를 쓰는가
  - 우리가 다룰 수 있는 빈틈은 무엇인가
```
