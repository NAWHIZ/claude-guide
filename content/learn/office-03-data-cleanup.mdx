# 지저분한 데이터, 5분 만에 정리하기

여러 사람이 입력한 엑셀 파일을 열면 형식이 제각각이에요. 날짜는 어디는 2024-01-01, 어디는 24/1/1, 이름은 어디는 "홍길동", 어디는 "홍 길동"... 이런 데이터 정제 작업을 Claude Code가 자동으로 해줍니다.

---

## 어떤 데이터 문제를 해결할 수 있나요?

| 문제 | 해결 방법 |
|------|---------|
| 날짜 형식 제각각 | YYYY-MM-DD로 통일 |
| 이름 앞뒤 공백 | 자동 제거 |
| 중복 데이터 | 중복 행 감지 및 제거 |
| 빈 셀 | 규칙에 따라 채우기 또는 표시 |
| 전화번호 형식 불일치 | 010-0000-0000으로 통일 |
| 대소문자 혼재 | 규칙에 따라 통일 |

---

## 1단계: 데이터 문제 진단

```
> customer-data.csv를 읽고 데이터 품질 문제를 진단해줘.
  
  확인 항목:
  1. 각 컬럼의 데이터 형식 불일치
  2. 빈 셀 위치와 비율
  3. 중복 행 수
  4. 형식 이상값 (날짜, 이메일, 전화번호)
  5. 의심스러운 값 (음수 나이, 미래 날짜 등)
  
  data-quality-report.md로 저장해줘.
```

---

## 2단계: 자동 정제

```
> customer-data.csv를 정제해줘.
  
  정제 규칙:
  - 이름: 앞뒤 공백 제거, 중간 공백 1개로 통일
  - 이메일: 소문자로 변환, 유효성 검사
  - 전화번호: 010-0000-0000 형식으로 통일
  - 날짜: YYYY-MM-DD 형식으로 통일
  - 중복 행: 제거 (이메일 기준)
  - 빈 셀: "미입력"으로 표시
  
  정제된 파일: customer-data-clean.csv
  변경 내역 로그: cleaning-log.txt
```

---

## 3단계: 데이터 분석 및 요약

정제된 데이터로 바로 분석을 시작해요:

```
> customer-data-clean.csv를 분석해줘.
  
  분석 항목:
  1. 총 고객 수
  2. 지역별/연령대별 분포
  3. 가입 월별 추이
  4. 활성 고객 vs 비활성 고객 비율
  5. 주요 인사이트 3가지
  
  analysis-summary.md로 저장.
```

---

## 4단계: 여러 파일 합치기

여러 부서에서 따로 관리하던 파일을 합칠 때:

```
> 다음 파일들을 하나로 합쳐줘:
  - team-a-data.csv
  - team-b-data.csv
  - team-c-data.csv
  
  처리:
  - 컬럼명이 다른 것은 매핑해서 통일
  - 중복 항목 제거 (이메일 기준)
  - 어느 파일에서 왔는지 'source' 컬럼 추가
  
  결과: combined-data.csv
```

---

## 5단계: 정기 데이터 정제 자동화

매달 같은 파일을 정제해야 한다면:

```
> 매달 받는 데이터 파일을 자동 정제하는 스크립트를 만들어줘.
  
  scripts/clean-monthly-data.py
  
  기능:
  1. data/raw/ 폴더에서 이번 달 파일 찾기
  2. 정제 규칙 적용 (config.json에서 읽기)
  3. 정제 결과를 data/clean/ 폴더에 저장
  4. 정제 요약 리포트 생성
  
  config.json에는 컬럼별 정제 규칙 저장.
```

---

## 실전 팁: 엑셀 VLOOKUP 자동화

```
> 두 파일을 특정 키로 연결해서 데이터를 합쳐줘.
  
  파일 1: orders.csv (주문 ID, 고객 ID 포함)
  파일 2: customers.csv (고객 ID, 이름, 이메일 포함)
  
  연결 키: 고객 ID
  결과: orders에 고객 이름과 이메일 컬럼 추가
  
  엑셀의 VLOOKUP과 같은 작업이야.
  결과: orders-enriched.csv
```

<Callout type="info">
파이썬을 몰라도 됩니다. Claude Code가 pandas 코드를 만들고 실행까지 해줘요. 데이터 파일만 있으면 돼요.
</Callout>
