# 훅(Hook)으로 자동 규칙 만들기

훅은 Claude Code가 특정 행동을 하기 전이나 후에 자동으로 뭔가를 실행하게 만드는 기능이에요. 한 번 설정해두면 매번 신경 쓰지 않아도 자동으로 동작합니다.

## 훅이 뭔가요?

> **비유**: 자동문 같은 거예요. 사람이 가까이 오면(특정 상황) 자동으로 문이 열리죠(자동 실행). 여러분이 매번 문을 열 필요가 없어요. 훅도 마찬가지로 "Claude Code가 파일을 수정할 때마다 자동으로 백업해줘" 같은 규칙을 만들 수 있어요.

훅의 종류:
- **PreToolUse**: Claude Code가 어떤 도구를 쓰기 **전에** 실행
- **PostToolUse**: Claude Code가 어떤 도구를 쓴 **후에** 실행
- **Stop**: Claude Code가 응답을 **완료한 후에** 실행

---

## 훅 설정하는 가장 쉬운 방법 — Claude Code에게 부탁하기

훅 설정은 Claude Code에게 말로 부탁하면 돼요. 설정 파일을 직접 건드릴 필요 없어요.

```
> ~/.claude/settings.json 파일에 훅을 추가해줘.
  Claude Code가 작업을 완료할 때마다 Mac 알림음이 나오게 해줘.
```

```
> 현재 프로젝트의 .claude/settings.json에 훅 추가해줘.
  Claude Code가 파일을 수정하기 전에 항상 .backups/ 폴더에 백업하게 해줘.
```

Claude Code가 알아서 설정 파일을 찾아서 추가해줘요.

---

## 실용적인 훅 예시

### 작업 완료 시 소리 알림 (가장 유용!)

긴 작업을 시켜놓고 다른 일 하다가 완료 소리를 들을 수 있어요.

```
> Claude Code 작업이 끝날 때마다 Mac 알림음이 나오게 훅 설정해줘
```

### 파일 수정 전 자동 백업

Claude Code가 파일을 수정하기 전에 자동으로 백업 폴더에 복사해둬요.

```
> Claude Code가 파일을 편집하기 전에 항상 .backups/ 폴더에 백업 저장하는 훅 만들어줘
```

### 데스크탑 알림

```
> Claude Code 작업 완료 시 Mac 데스크탑 알림이 뜨게 해줘
```

### 위험한 명령어 자동 차단

실수로 중요한 파일을 삭제하는 명령어를 자동으로 막아줘요.

```
> 파일을 강제 삭제하는 명령어(rm -rf 등)를 Claude Code가 실행하려 하면
  자동으로 차단하는 훅 만들어줘
```

---

## 훅이 설정되는 위치

훅은 두 곳에 저장될 수 있어요:

| 위치 | 설정 파일 | 적용 범위 |
|------|-----------|-----------|
| 전역 | `~/.claude/settings.json` | 모든 프로젝트 |
| 프로젝트 | `.claude/settings.json` | 현재 프로젝트만 |

소리 알림처럼 어디서나 쓰고 싶은 건 전역에, 특정 프로젝트 백업 규칙은 프로젝트 설정에 넣으면 돼요.

---

## 설정된 훅 확인/제거하기

```
> 현재 설정된 훅 목록을 알려줘
```

```
> 파일 백업 훅을 제거해줘
```

<Callout type="warning">
훅에 잘못된 설정이 있으면 Claude Code 작동에 문제가 생길 수 있어요. 이상이 생기면 "훅 설정 전부 지워줘"라고 요청하세요.
</Callout>

<Callout type="tip">
OMC(oh-my-claudecode)를 설치하면 유용한 기본 훅들이 자동으로 설정돼요. 직접 설정하는 것보다 OMC 설치 후 쓰는 게 더 간편해요..
</Callout>
