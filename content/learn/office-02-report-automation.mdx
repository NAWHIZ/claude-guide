# 월간 보고서, 버튼 하나로 만들기

매달 말 엑셀 데이터를 보고 파워포인트를 만드는 데 하루를 쓰고 계신가요? 데이터 파일만 주면 Claude Code가 완성된 보고서를 만들어줍니다.

---

## 어떤 보고서를 자동화할 수 있나요?

- 월간 실적 보고서
- 부서별 업무 현황 보고서
- 프로젝트 진행 현황 보고서
- 예산 집행 보고서
- KPI 달성 현황 보고서

---

## 1단계: 데이터 파일 준비

엑셀(.xlsx) 또는 CSV 파일을 Claude Code와 같은 폴더에 저장해요.

예시 파일들:
- `monthly-sales.csv` (월별 매출 데이터)
- `project-status.xlsx` (프로젝트 현황)
- `budget-usage.csv` (예산 집행 현황)

---

## 2단계: 보고서 초안 생성

```
> monthly-sales.csv와 project-status.xlsx를 읽고
  월간 업무 보고서를 만들어줘.
  
  보고서 구성:
  1. 이번 달 핵심 성과 요약 (3줄 이내)
  2. 주요 지표 (목표 vs 실적 표)
  3. 이전 달 대비 변화 (% 표시)
  4. 주목할 이슈 또는 리스크
  5. 다음 달 계획
  
  대상: 팀장/임원 보고용
  형식: monthly-report-[월].md
```

---

## 3단계: HTML 보고서로 변환

```
> monthly-report-[월].md를 전문적인 HTML 보고서로 변환해줘.
  
  디자인 요구사항:
  - 회사 로고 포함 (logo.png 파일 있으면 사용)
  - 깔끔한 테이블 스타일
  - 차트 (recharts 또는 Chart.js)
  - 인쇄 가능한 A4 레이아웃
  - 회사 브랜드 색상: [색상 코드]
  
  monthly-report-[월].html로 저장.
```

---

## 4단계: PDF 변환

```
> monthly-report-[월].html을 PDF로 변환해줘.
  Chrome Headless 사용.
  
  저장 위치: reports/monthly-report-[월].pdf
```

Mac에 Chrome이 설치되어 있으면 별도 설치 없이 바로 실행돼요.

---

## 5단계: 보고서 자동화 스크립트 만들기

매달 반복 실행하는 과정을 스크립트로 만들어요:

```
> 월간 보고서 생성을 자동화하는 스크립트를 만들어줘.
  
  scripts/generate-monthly-report.sh
  
  실행 단계:
  1. data/ 폴더에서 이번 달 데이터 파일 확인
  2. Claude API로 보고서 내용 생성
  3. HTML 템플릿에 데이터 삽입
  4. PDF 변환
  5. reports/ 폴더에 저장
  6. Slack으로 완료 알림
  
  실행 방법: ./generate-monthly-report.sh 2026-03
```

---

## 실전 팁: 여러 보고서 형식 관리

부서마다 다른 형식이 필요해요:

```
> 다음 3가지 보고서 템플릿을 만들어줘:
  
  1. 임원 요약 보고서 (1페이지, 핵심 수치 + 그래프)
  2. 팀 상세 보고서 (3-5페이지, 상세 내용)
  3. 주간 스냅샷 (이메일 형식, 3개 수치 + 1개 이슈)
  
  templates/reports/ 폴더에 저장.
  각 템플릿은 변수 [괄호]로 표시.
```

<BeforeAfter>
  <Before>
    데이터 정리(2h) + 파워포인트 작성(3h) + 표/차트 작업(2h) = 7시간
  </Before>
  <After>
    데이터 파일 준비(30분) + 스크립트 실행(5분) + 검토(30분) = 65분
  </After>
</BeforeAfter>
