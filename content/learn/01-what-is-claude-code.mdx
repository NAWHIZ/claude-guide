# Claude Code가 뭔가요?

Claude Code는 AI 회사 Anthropic이 만든 도구예요. 쉽게 말하면, 여러분의 컴퓨터 파일을 직접 열고, 읽고, 수정할 수 있는 AI 비서입니다.

## 터미널에서 실행되는 도구

Claude Code는 **터미널(Terminal)** 이라는 곳에서 실행돼요. 터미널이 생소하다면, 컴퓨터에 텍스트로 명령을 내리는 검은 창이라고 생각하시면 됩니다. Mac에서는 "터미널(Terminal.app)", Windows에서는 "PowerShell"이라고 불리는 프로그램이에요.

터미널이 무섭게 느껴지실 수 있어요. 하지만 Claude Code를 쓰기 위해 터미널에서 해야 할 일은 딱 하나예요 — `claude`라고 입력하고 엔터를 누르는 것. 그 다음부터는 평소 채팅하듯이 한국어로 대화하면 됩니다.

## Claude.ai 웹 버전과 무엇이 다른가요?

많은 분들이 이미 [Claude.ai](https://claude.ai) 웹사이트에서 Claude를 사용해보셨을 거예요. 웹 버전과 Claude Code의 가장 큰 차이는 딱 한 가지입니다.

**Claude.ai 웹 버전**: 대화만 할 수 있어요. 파일 내용을 붙여넣기 해서 보여줄 수는 있지만, Claude가 직접 여러분의 파일을 열거나 수정할 수는 없어요.

**Claude Code**: 여러분 컴퓨터의 파일을 **직접** 열고, 읽고, 수정하고, 새로 만들 수 있어요. "이 파일 수정해줘"라고 하면 실제로 파일이 바뀝니다.

이 차이가 Claude Code를 훨씬 강력하게 만들어줘요.

<Callout type="tip">
Claude Code는 여러분의 파일을 실제로 열고 수정할 수 있어요. 웹 버전처럼 대화만 하는 것이 아니라, 진짜로 컴퓨터 파일에 손을 댑니다. 그래서 훨씬 더 많은 일을 자동으로 처리해줄 수 있어요.
</Callout>

## 비개발자가 Claude Code를 쓰면 좋은 이유

"저는 코딩을 전혀 모르는데요?"라고 생각하실 수 있어요. 전혀 문제없습니다. Claude Code를 쓰기 위해 코딩을 알 필요가 없어요. 오히려 **코딩을 모르는 분들**에게 더 유용할 수 있습니다.

왜냐하면 이런 일들을 대신 해주기 때문이에요:

- **파일을 직접 수정해줘요**: "이 문서의 1번 항목을 2번으로 바꿔줘"처럼 말하면 됩니다
- **코드를 대신 써줘요**: HTML, CSS, Python 등 코드를 몰라도 "간단한 웹페이지 만들어줘"라고 하면 파일을 만들어줘요
- **반복 작업을 자동화해줘요**: 100개 파일 이름을 한꺼번에 바꾸거나, 여러 문서를 한 번에 정리하는 일을 해줘요
- **설명해줘요**: "이 파일이 뭘 하는 건지 설명해줘"라고 하면 이해하기 쉽게 알려줘요

## Claude Code로 할 수 있는 것들

실제로 어떤 일들을 할 수 있는지 구체적으로 살펴볼게요.

### 파일 읽기와 요약
"이 폴더에 있는 모든 텍스트 파일을 읽고 핵심 내용을 요약해줘"라고 하면, Claude Code가 파일들을 하나씩 열어서 내용을 정리해줍니다.

### 파일 수정
"index.html 파일에서 회사 이름을 '구회사'에서 '새회사'로 바꿔줘"라고 하면, 파일을 직접 열어서 수정합니다.

### 폴더 탐색
"이 프로젝트 폴더 안에 어떤 파일들이 있는지 설명해줘"라고 하면 폴더 구조를 파악해서 설명해줍니다.

### 코드 설명
개발자가 만든 코드 파일이 있다면 "이 코드가 어떤 역할을 하는지 쉽게 설명해줘"라고 할 수 있어요.

### 새 파일 만들기
"오늘 회의 내용을 정리한 meeting-notes.md 파일 만들어줘"라고 하면 파일을 새로 생성해줍니다.

### 반복 작업
"이 폴더 안의 모든 .txt 파일 이름 앞에 날짜를 붙여줘"처럼 반복적인 파일 작업을 한 번에 처리해줘요.
