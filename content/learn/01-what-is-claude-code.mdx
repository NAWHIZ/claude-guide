# Claude Code가 뭔가요?

Claude Code는 AI 회사 Anthropic이 만든 도구예요. 쉽게 말하면, 여러분의 컴퓨터 파일을 직접 열고, 읽고, 수정할 수 있는 AI 비서입니다.

## 터미널에서 실행되는 도구

Claude Code는 **터미널(Terminal)** 이라는 곳에서 실행돼요. 터미널이 뭔지 모르시는 분이 많을 텐데, 쉽게 말하면 "컴퓨터에 말을 걸 수 있는 특별한 창"이에요.

> **비유**: 스마트폰 Siri나 빅스비에게 말을 걸 듯, 터미널에서는 텍스트로 컴퓨터에게 명령을 내려요. 화려한 버튼이나 아이콘 대신 텍스트를 입력하는 방식이에요.

Mac에서는 "터미널(Terminal.app)", Windows에서는 "PowerShell"이라고 불리는 프로그램이에요.

터미널이 무섭게 느껴지실 수 있어요. 하지만 Claude Code를 쓰기 위해 터미널에서 해야 할 일은 딱 하나예요 — `claude`라고 입력하고 엔터를 누르는 것. 그 다음부터는 평소 채팅하듯이 한국어로 대화하면 됩니다.

## Claude.ai 웹 버전과 무엇이 다른가요?

많은 분들이 이미 [Claude.ai](https://claude.ai) 웹사이트에서 Claude를 사용해보셨을 거예요. 웹 버전과 Claude Code의 가장 큰 차이는 딱 한 가지입니다.

**Claude.ai 웹 버전**: 대화만 할 수 있어요. 파일 내용을 붙여넣기 해서 보여줄 수는 있지만, Claude가 직접 여러분의 파일을 열거나 수정할 수는 없어요.

**Claude Code**: 여러분 컴퓨터의 파일을 **직접** 열고, 읽고, 수정하고, 새로 만들 수 있어요. "이 파일 수정해줘"라고 하면 실제로 파일이 바뀝니다.

이 차이가 Claude Code를 훨씬 강력하게 만들어줘요.

<Callout type="tip">
Claude Code는 여러분의 파일을 실제로 열고 수정할 수 있어요. 웹 버전처럼 대화만 하는 것이 아니라, 진짜로 컴퓨터 파일에 손을 댑니다. 그래서 훨씬 더 많은 일을 자동으로 처리해줄 수 있어요.
</Callout>

## 코딩을 몰라도 되는 이유

"저는 코딩을 전혀 모르는데요?"라고 생각하실 수 있어요. 전혀 문제없습니다.

Claude Code는 **코딩 언어가 아닌 한국어로 대화**해요. 마치 일 잘하는 비서에게 "이 파일 좀 정리해줘"라고 말하듯이 사용하면 됩니다. 코딩은 Claude Code가 알아서 해요.

왜냐하면 이런 일들을 대신 해주기 때문이에요:

- **파일을 직접 수정해줘요**: "이 문서의 1번 항목을 2번으로 바꿔줘"처럼 말하면 됩니다
- **코드를 대신 써줘요**: HTML, CSS, Python 등 코드를 몰라도 "간단한 웹페이지 만들어줘"라고 하면 파일을 만들어줘요
- **반복 작업을 자동화해줘요**: 100개 파일 이름을 한꺼번에 바꾸거나, 여러 문서를 한 번에 정리하는 일을 해줘요
- **설명해줘요**: "이 파일이 뭘 하는 건지 설명해줘"라고 하면 이해하기 쉽게 알려줘요

## Claude Code로 할 수 있는 것들

실제로 어떤 일들을 할 수 있는지 구체적으로 살펴볼게요.

### 📄 파일 읽기와 요약

```
> 이 폴더에 있는 모든 회의록 파일을 읽고 핵심 결정사항만 정리해줘
```

```
> report.pdf 내용을 3줄로 요약해줘
```

```
> 이 엑셀 파일에서 3월 매출이 가장 높은 상품 5개를 알려줘
```

Claude Code가 파일들을 직접 열어서 내용을 분석하고, 여러분이 원하는 형태로 정리해줍니다.

### ✏️ 파일 수정

```
> index.html 파일에서 회사 이름을 '구회사'에서 '새회사'로 모두 바꿔줘
```

```
> 이 폴더의 모든 문서에서 작년 전화번호를 새 번호로 바꿔줘
```

```
> 계약서 templates 폴더 안 파일들에서 '갑'을 '발주처'로 일괄 변경해줘
```

원하는 내용을 말하면 Claude Code가 파일을 직접 열어서 수정합니다. 10개든 100개든 한 번에 처리해요.

### 📁 폴더 정리와 파악

```
> 이 프로젝트 폴더 안에 어떤 파일들이 있는지 설명해줘
```

```
> Downloads 폴더에서 6개월 이상 된 파일 목록을 보여줘
```

```
> 이 폴더 안의 파일들을 날짜별로 정리해서 연도/월 폴더로 옮겨줘
```

복잡한 폴더 구조도 순식간에 파악하고, 원하는 방식으로 정리해줍니다.

### 🆕 새 파일 만들기

```
> 오늘 회의 내용을 정리할 수 있는 회의록 템플릿 파일 만들어줘
```

```
> 이 데이터를 기반으로 월간 보고서 초안을 작성해줘
```

```
> 직원 목록 CSV 파일을 만들어줘 — 이름, 부서, 이메일, 입사일 컬럼으로
```

### 🌐 웹사이트 파일 수정 (HTML)

코딩을 몰라도 웹사이트 파일을 수정할 수 있어요:

```
> 웹사이트 모든 페이지 하단에 '© 2025 회사명' 문구를 추가해줘
```

```
> contact.html의 전화번호와 주소를 새것으로 바꿔줘
```

```
> 회사 소개 페이지에 신제품 배너를 추가해줘
```

### 🔄 반복 작업 자동화

사람이 하면 몇 시간 걸릴 일도 Claude Code에게는 몇 분이에요:

```
> 이 폴더 안의 모든 이미지 파일 이름 앞에 '20250326_'을 붙여줘
```

```
> 모든 보고서 파일을 읽고 각각의 핵심 요약을 한 파일에 모아줘
```

```
> 고객 명단 파일에서 서울 지역 고객만 골라 새 파일로 만들어줘
```

### 🤝 개발자가 만든 것 이해하기

개발자가 만들어 준 파일이나 코드가 뭔지 이해해야 할 때:

```
> 개발자가 보내준 이 파일들이 어떤 역할을 하는지 쉽게 설명해줘
```

```
> 이 설정 파일에서 서버 주소 바꾸는 부분이 어디야?
```

```
> 이 에러 메시지가 무슨 뜻인지, 어떻게 해야 하는지 알려줘
```

## 실제로 어떤 분들이 많이 쓰고 있나요?

**마케터/기획자**: 대량 문서 정리, 보고서 초안 작성, 데이터 파일 분석

**소규모 사업자**: 웹사이트 텍스트 수정, 상품 설명 파일 일괄 업데이트, 고객 데이터 정리

**연구자/학생**: 논문 자료 정리, 데이터 파일 처리, 참고문헌 정리

**팀장/매니저**: 여러 팀원의 보고서 취합 및 요약, 회의록 관리

코딩을 한 줄도 모르는 분들이 이미 실무에서 잘 쓰고 있어요.

실제로 어떤 일들을 할 수 있는지 구체적으로 살펴볼게요.

### 파일 읽기와 요약
"이 폴더에 있는 모든 텍스트 파일을 읽고 핵심 내용을 요약해줘"라고 하면, Claude Code가 파일들을 하나씩 열어서 내용을 정리해줍니다.

### 파일 수정
"index.html 파일에서 회사 이름을 '구회사'에서 '새회사'로 바꿔줘"라고 하면, 파일을 직접 열어서 수정합니다.

### 폴더 탐색
"이 프로젝트 폴더 안에 어떤 파일들이 있는지 설명해줘"라고 하면 폴더 구조를 파악해서 설명해줍니다.

### 코드 설명
개발자가 만든 코드 파일이 있다면 "이 코드가 어떤 역할을 하는지 쉽게 설명해줘"라고 할 수 있어요.

### 새 파일 만들기
"오늘 회의 내용을 정리한 meeting-notes.md 파일 만들어줘"라고 하면 파일을 새로 생성해줍니다.

### 반복 작업
"이 폴더 안의 모든 .txt 파일 이름 앞에 날짜를 붙여줘"처럼 반복적인 파일 작업을 한 번에 처리해줘요.
