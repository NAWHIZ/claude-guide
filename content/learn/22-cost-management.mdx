# Claude Code 비용 관리하기

Claude Code를 쓰다 보면 "이거 돈이 얼마나 나오지?"가 걱정돼요. 미리 구조를 이해해두면 예상치 못한 청구서를 받는 일이 없어요.

---

## Claude Code 요금 구조

Claude Code는 **Claude.ai 구독 플랜**을 기반으로 해요.

| 플랜 | 월 요금 | 특징 |
|------|---------|------|
| **Free** | 무료 | 하루 사용량 제한 있음 |
| **Pro** | $20/월 (약 3만원) | 일반 사용자에게 충분 |
| **Max** | $100/월~ | 헤비 유저, 팀 사용 |

> **비유**: 넷플릭스 구독처럼, 월정액을 내면 정해진 범위 내에서 마음껏 쓸 수 있어요. Pro 플랜이면 일반적인 업무에 충분해요.

<Callout type="info">
Claude Code는 claude.ai 구독에 포함돼요. claude.ai에서 Pro 플랜을 결제하면 Claude Code도 같이 쓸 수 있어요.
</Callout>

---

## 실제로 얼마나 드나요?

**Pro 플랜 ($20/월)으로 할 수 있는 것:**
- 하루 수십 번 이상 파일 읽기/수정
- 웹서비스 1-2개 구축
- 매일 업무 자동화 작업

**현실적인 월 비용 시나리오:**

| 사용 패턴 | 추천 플랜 | 월 비용 |
|-----------|-----------|---------|
| 가끔 써보기 | Free | 무료 |
| 업무에 매일 사용 | Pro | $20 (약 3만원) |
| 팀 전체 / 대형 프로젝트 | Max | $100~ |

---

## 추가로 드는 비용들

Claude Code 외에 서비스를 만들 때 드는 비용이에요.

| 서비스 | 무료 한도 | 유료 시작 |
|--------|-----------|-----------|
| **Vercel** (배포) | 취미 프로젝트 무료 | $20/월 |
| **Supabase** (DB) | 500MB, 소규모 무료 | $25/월 |
| **GitHub** (코드 저장) | 공개/비공개 무료 | - |
| **Tavily** (웹검색 MCP) | 월 1,000회 무료 | $20/월 |
| **Notion** (문서) | 기본 무료 | $10/월 |

**결론**: 시작할 때는 Claude Code Pro($20) + Vercel 무료 + Supabase 무료로 총 **월 $20**이면 충분해요.

## 비용을 줄이는 실전 팁

### 1. `/compact`로 대화 압축하기

대화가 길어질수록 처리할 내용이 많아져서 한도를 빨리 소진해요. 중간중간 압축해주면 효율이 올라가요.

```
> /compact
```

### 2. 새 작업은 `/clear`로 시작하기

다른 주제로 넘어갈 때 이전 대화를 지우면 낭비 없이 새로 시작할 수 있어요.

```
> /clear
```

### 3. 간단한 작업은 Haiku 모델 사용

파일 이름 바꾸기, 간단한 텍스트 수정 같은 작업은 가벼운 모델을 써도 충분해요.

```
> /model
```
→ Haiku 선택 (더 빠르고 저렴)

### 4. 작업을 구체적으로 요청하기

모호한 요청은 Claude Code가 여러 번 시도하게 만들어요.

```
❌ "이거 좀 고쳐줘"
✅ "report.md 파일의 3번째 섹션 제목을 '분기별 실적'으로 바꿔줘"
```

### 5. 큰 작업은 단계별로 나누기

한 번에 너무 많은 걸 요청하면 중간에 실패했을 때 처음부터 다시 해야 해요.

```
> 1단계: 먼저 파일 구조만 만들어줘
> 2단계: 이제 내용을 채워줘
> 3단계: GitHub에 올려줘
```

---

## 한도에 걸렸을 때

Claude Code가 "한도에 도달했다"고 하면:

1. **잠깐 기다리기** — 시간이 지나면 리셋돼요 (보통 몇 시간)
2. **다음날 다시 시도** — 일 단위로 초기화돼요
3. **Max 플랜 고려** — 매일 한도에 걸린다면 업그레이드

<Callout type="tip">
Pro 플랜 한도는 생각보다 넉넉해요. 일반적인 업무 사용자라면 하루에 한도를 다 쓰기 어렵습니다. 처음엔 Pro로 시작해서 필요하면 올리세요.
</Callout>

## 비용 관련 자주 묻는 질문

**Q: 무료로 시작해도 되나요?**
Free 플랜으로 시작해서 한도에 자주 걸리면 그때 Pro로 올리면 돼요.

**Q: Claude Code와 claude.ai 구독이 별개인가요?**
아니요, 같아요. claude.ai에서 Pro 플랜을 결제하면 claude.ai 웹과 Claude Code CLI 모두 사용할 수 있어요.

**Q: Vercel, Supabase 비용이 갑자기 나올 수 있나요?**
무료 한도를 초과하면 청구될 수 있어요. Vercel과 Supabase 대시보드에서 사용량을 확인할 수 있고, 예산 알림을 설정하면 갑작스러운 청구를 막을 수 있어요.

**Q: 팀원들과 함께 쓰려면?**
각자 Claude Code 계정이 필요해요. 공유 계정은 권장하지 않아요 (보안 및 한도 문제).
