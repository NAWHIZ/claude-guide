# 경쟁사 분석, 매주 자동으로

PM이 가장 자주 받는 질문 중 하나: "경쟁사는 요즘 뭐 하고 있어요?" 이 질문에 대한 답을 항상 준비하려면 지금까지는 매주 직접 서치해야 했어요. 이제는 Claude Code + Tavily MCP가 자동으로 해줍니다.

<Callout type="tip">
이 자동화가 완성되면 매주 월요일 아침에 경쟁사 동향 브리핑이 Notion에 자동으로 올라와요.
</Callout>

---

## 필요한 것

| 도구 | 역할 | 준비 |
|------|------|------|
| Tavily MCP | 웹 검색 자동화 | [MCP 설정](/learn/13-mcp-servers) |
| Notion MCP | 결과 저장 | [MCP 설정](/learn/13-mcp-servers) |
| Claude Code | 분석 + 정리 | 기본 설치 |

---

## 1단계: 경쟁사 1개 즉시 분석

먼저 단발성으로 경쟁사 분석을 해봐요:

```
> Tavily로 [경쟁사명]의 최근 동향을 검색해줘.
  
  검색 키워드:
  - "[경쟁사명] 신기능"
  - "[경쟁사명] 업데이트 2025"
  - "[경쟁사명] pricing"
  
  결과를 다음 형식으로 정리해줘:
  
  ## 최신 업데이트 (날짜 포함)
  ## 신규 기능
  ## 가격 변동
  ## 마케팅 메시지 변화
  ## 시사점 (우리 제품에 미치는 영향)
```

---

## 2단계: 복수 경쟁사 비교 분석

```
> Tavily로 다음 경쟁사들의 최근 1개월 동향을 검색해줘:
  - 경쟁사 A
  - 경쟁사 B
  - 경쟁사 C
  
  각 경쟁사별로 정리한 후, 마지막에 비교 테이블을 추가해줘.
  
  비교 항목: 주요 업데이트, 가격, 강조하는 기능
  
  competitor-analysis-[오늘날짜].md로 저장해줘.
```

---

## 3단계: Notion 자동 업데이트

```
> competitor-analysis-[날짜].md 내용을
  Notion "경쟁사 분석" 데이터베이스에 새 페이지로 저장해줘.
  
  속성:
  - 날짜: 오늘
  - 분석 유형: 정기 업데이트
  - 주요 발견사항: [핵심 인사이트 1줄]
```

---

## 4단계: 자동화 스크립트 만들기

이 과정을 매주 자동으로 실행하려면:

```
> 경쟁사 모니터링을 자동화하는 스크립트를 만들어줘.
  
  파일: scripts/competitor-monitor.sh
  
  기능:
  1. 경쟁사 목록 (config에서 읽기)
  2. Tavily로 각 경쟁사 검색
  3. Claude API로 분석 및 정리
  4. Notion에 자동 저장
  
  설정 파일: competitor-config.json
  (경쟁사 이름, 검색 키워드 목록 포함)
```

---

## 5단계: 자동 알림 추가

중요한 변화가 있을 때만 알림을 받을 수 있어요:

```
> 경쟁사 분석 결과에서 다음 중 하나라도 해당하면
  Slack #product 채널에 즉시 알림을 보내줘:
  
  - 가격 변경
  - 새 기능 출시
  - 우리 제품과 직접 경쟁하는 내용
  
  Slack MCP가 필요해. 설정되어 있으면 연동해줘.
```

---

## 경쟁사 인텔리전스 데이터베이스 만들기

시간이 지나면서 쌓이는 데이터를 활용해요:

```
> 지금까지 쌓인 competitor-analysis-*.md 파일들을 읽고
  경쟁사별 변화 추이를 분석해줘.
  
  출력:
  - 각 경쟁사의 전략 방향 변화
  - 가격 변화 트렌드
  - 기능 발전 패턴
  - 우리가 놓치고 있는 기회
```

<Callout type="info">
분기에 한 번씩 이 트렌드 분석을 실행하면 경쟁사의 장기 전략을 읽을 수 있어요.
</Callout>
