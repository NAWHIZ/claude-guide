# 자동화와 고급 활용

Claude Code를 어느 정도 사용해봤다면, 이제 더 강력하게 활용하는 방법을 배워볼게요. 복잡한 작업을 자동화하고, 여러 단계를 한 번에 처리하는 방법을 알아봅니다.

## 멀티 스텝 작업

여러 단계를 한 번에 요청할 수 있어요:

```
> 다음을 순서대로 해줘:
  1. reports/ 폴더의 모든 파일을 읽어
  2. 각 파일에서 "결론" 섹션만 추출해
  3. 추출한 내용을 monthly_conclusions.md로 저장해
```

```
> sales_2023.csv와 sales_2024.csv를 읽고,
  연도별 매출 증감을 계산한 다음
  comparison_report.md로 요약 보고서를 만들어줘
```

## 조건부 작업

조건을 걸어서 작업할 수도 있어요:

```
> invoices/ 폴더의 모든 파일을 확인해서
  금액이 100만원 이상인 청구서만 high_value/ 폴더로 복사해줘
```

```
> 파일 수정일이 30일 이상 된 .tmp 파일을 찾아서
  목록을 보여줘 (삭제는 하지 마)
```

## 반복 패턴 작업

같은 패턴이 반복되는 작업은 Claude Code가 탁월해요:

```
> templates/email_template.txt 파일을 읽고
  contacts.csv의 각 사람 이름으로 개인화된 이메일을 만들어
  emails/ 폴더에 이름별로 저장해줘
```

```
> 이 폴더의 모든 마크다운 파일에서
  제목(# 으로 시작하는 첫 줄)을 추출해서
  목차 파일 table_of_contents.md를 만들어줘
```

## 데이터 변환 작업

다양한 형식 간 변환도 가능해요:

```
> data.csv 파일을 읽고
  이메일, 이름, 연락처 컬럼만 추출해서
  contacts_clean.csv로 저장해줘
```

```
> 이 JSON 파일의 내용을 읽기 쉬운 마크다운 표로 변환해줘
```

## 프로젝트 분석과 리포팅

```
> 이 프로젝트 폴더 전체를 분석해서
  - 총 파일 수와 용량
  - 파일 형식별 분류
  - 가장 최근에 수정된 파일 10개
  를 담은 project_audit.md를 만들어줘
```

```
> git log를 읽고 지난 한 달간의 변경 사항을 요약해줘
  (개발 프로젝트용)
```

## 커스텀 워크플로우 만들기

자주 하는 작업 흐름을 CLAUDE.md에 정의해두면 매번 길게 설명하지 않아도 돼요:

```markdown
## 주간 보고서 워크플로우

"주간 보고서 만들어줘"라고 하면:
1. weekly_data/ 폴더의 이번 주 파일들을 모두 읽는다
2. 완료된 작업, 진행 중인 작업, 이슈 사항을 분류한다
3. reports/YYYYWW_weekly_report.md 형식으로 저장한다
4. 보고서 완성 후 내용 요약을 알려준다
```

이렇게 설정해두면:

```
> 주간 보고서 만들어줘
```

한 줄로 전체 워크플로우가 실행돼요.

## 다른 도구와 함께 사용하기

Claude Code는 시스템 명령어도 실행할 수 있어요:

```
> 현재 폴더에서 가장 큰 파일 10개를 찾아줘
```

```
> 오늘 수정된 파일 목록을 보여줘
```

```
> PDF 파일 목록을 만들고 각 파일 크기도 함께 보여줘
```

## 오류 대처하기

Claude Code가 예상대로 작동하지 않을 때:

**응답이 길어지면**: `/compact`로 대화 압축 후 계속하기

**엉뚱한 작업을 시작하면**: `Ctrl + C`로 즉시 중단

**실수로 파일을 잘못 수정했으면**:
```
> 방금 수정한 파일을 원래대로 되돌려줘
```

**작업이 너무 복잡하면**: 작은 단계로 나눠서 요청하기

<Callout type="tip">
복잡한 자동화 작업은 처음에 테스트용 파일 몇 개로만 시도해보세요. 결과가 만족스러우면 전체 폴더에 적용하는 게 안전해요.
</Callout>

## 고급 프롬프트 작성 팁

더 정확한 결과를 얻으려면:

1. **맥락 제공**: "마케팅 팀에서 사용하는 보고서야" 처럼 배경 설명
2. **예시 포함**: "이런 형식으로 해줘: [예시]"
3. **제약 명시**: "반드시 한국어로", "100줄 이하로"
4. **단계 분리**: 한 번에 너무 많은 작업 요청 피하기
5. **확인 요청**: "실행 전에 계획을 먼저 알려줘"

Claude Code를 꾸준히 사용하다 보면, 어느새 "이걸 Claude한테 맡기면 되겠다"는 감각이 생겨요. 그 순간부터 여러분의 업무 생산성이 크게 달라집니다.
