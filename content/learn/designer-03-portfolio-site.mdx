# 디자이너 포트폴리오 사이트 하루 만에 만들기

노션으로 포트폴리오를 공유하던 시대는 끝났어요. 디자이너라면 포트폴리오 사이트 자체가 작업물이에요. Claude Code를 쓰면 코딩 없이 하루 만에 나만의 포트폴리오 사이트를 만들 수 있어요.

<Callout type="tip">
이 사이트(claude-guide)도 같은 방식으로 만들어졌어요. Vercel에 배포하면 `내이름.vercel.app` 주소가 생기고, 커스텀 도메인도 연결할 수 있어요.
</Callout>

---

## 1단계: 포트폴리오 내용 정리

먼저 어떤 내용을 담을지 정리해요:

```
> 디자이너 포트폴리오 사이트 구조를 설계해줘.
  
  나에 대해:
  - 이름: [이름]
  - 전문 분야: UX/UI 디자인, 모바일 앱
  - 경력: [년]
  
  사이트에 담을 것:
  - 히어로 섹션 (소개 + CTA)
  - 프로젝트 갤러리 (6개)
  - 스킬셋
  - 경력 타임라인
  - 연락처
  
  구조도와 각 섹션 상세 내용 목록을 알려줘.
```

---

## 2단계: 사이트 생성

```
> Next.js로 디자이너 포트폴리오 사이트를 만들어줘.
  
  디자인 방향:
  - 미니멀하고 고급스러운 느낌
  - 배경: 거의 흰색 (#FAFAFA)
  - 강조색: 내 브랜드 컬러 (#[내 색상 코드])
  - 폰트: Pretendard (한국어), Inter (영어)
  
  인터랙션:
  - 스크롤 시 섹션 페이드인
  - 프로젝트 카드 hover 줌
  - 부드러운 페이지 전환
  
  반응형: 모바일 우선
```

---

## 3단계: 프로젝트 케이스 스터디 페이지

```
> 각 프로젝트의 케이스 스터디 페이지를 만들어줘.
  
  페이지 구성:
  1. 프로젝트 커버 이미지
  2. 개요 (클라이언트, 기간, 역할)
  3. 문제 정의
  4. 리서치 과정
  5. 디자인 과정 (스케치 → 와이어프레임 → 프로토타입)
  6. 최종 결과물 갤러리
  7. 결과 및 임팩트
  
  데이터는 /content/projects/ 폴더의 MDX 파일에서 읽어와.
  새 프로젝트를 추가할 때 MDX만 만들면 자동으로 페이지 생성.
```

---

## 4단계: 이미지 갤러리 최적화

```
> 포트폴리오 이미지가 빠르게 로드되게 최적화해줘.
  
  - Next.js Image 컴포넌트 사용
  - lazy loading
  - blur placeholder (로딩 중 흐릿하게)
  - 모바일에서 적절한 크기로 자동 조정
  
  이미지는 /public/portfolio/ 폴더에 넣으면 자동으로 사용.
```

---

## 5단계: SEO 최적화

취업 또는 프리랜서 영업을 위해 검색 노출이 중요해요:

```
> 포트폴리오 사이트 SEO를 최적화해줘.
  
  - 각 페이지 메타태그 (title, description)
  - Open Graph 이미지 (SNS 공유 시 미리보기)
  - 구조화된 데이터 (JSON-LD)
  - sitemap.xml 자동 생성
  - robots.txt
  
  타겟 키워드: "UX 디자이너 포트폴리오", "[내 이름]"
```

---

## 6단계: 연락처 폼 추가

```
> 연락처 폼을 추가해줘.
  
  필드: 이름, 이메일, 프로젝트 종류, 메시지
  
  제출하면:
  1. Supabase에 문의 내용 저장
  2. 내 이메일로 알림 발송
  3. 제출자에게 자동 응답 이메일
  
  이메일 발송은 Resend API 사용.
```

---

## 7단계: Vercel 배포

```
> 사이트를 Vercel에 배포해줘.
  환경변수:
  - NEXT_PUBLIC_SUPABASE_URL
  - NEXT_PUBLIC_SUPABASE_ANON_KEY
  - RESEND_API_KEY
```

GitHub에 푸시하면 자동 배포. `[프로젝트명].vercel.app` 주소로 접근 가능해요.

---

## 커스텀 도메인 연결

```
> 커스텀 도메인을 Vercel에 연결하는 방법을 알려줘.
  도메인: [내도메인.com]
```

<Callout type="info">
도메인은 가비아, 후이즈 같은 국내 업체 또는 Namecheap, Google Domains에서 구매할 수 있어요. 1년에 1-2만원 수준이에요.
</Callout>
