# 회의록, AI가 5분 만에 정리해줘

회의가 끝나고 회의록 쓰느라 30분을 써본 적 있으세요? 메모한 내용을 정리하고, 액션 아이템을 뽑고, 담당자를 지정하고... Claude Code는 이 과정을 5분으로 줄여줍니다.

<Callout type="tip">
녹음을 텍스트로 변환하려면 Clova Note(무료), VREW(무료) 또는 Whisper를 사용하세요. 텍스트만 있으면 Claude Code가 나머지를 다 해줘요.
</Callout>

---

## 1단계: 기본 회의록 정리

회의 메모나 텍스트를 그대로 붙여넣어요:

```
> 다음 회의 내용을 정리해줘:
  
  [회의 내용 붙여넣기 — 날것 그대로 OK]
  
  정리 형식:
  ## 회의 기본 정보
  - 날짜, 참석자, 주제
  
  ## 핵심 결정 사항
  - 결정된 것들만 (토론 과정 제외)
  
  ## 액션 아이템
  | 담당자 | 할 일 | 기한 |
  
  ## 다음 회의 주제
  
  meeting-notes-[날짜].md로 저장해줘.
```

---

## 2단계: Notion에 자동 저장

```
> meeting-notes-[날짜].md를 
  Notion "회의록" 데이터베이스에 저장해줘.
  
  속성:
  - 날짜: [회의 날짜]
  - 회의 유형: [정기/긴급/프로젝트별]
  - 참석자: [이름들]
  - 액션 아이템 완료 여부: 미완료
```

---

## 3단계: 담당자 자동 알림

```
> meeting-notes-[날짜].md의 액션 아이템을 읽고
  담당자별로 정리해서 Slack DM을 보내줘.
  
  메시지 형식:
  "안녕하세요 [이름]님,
  오늘 회의에서 담당하시게 된 항목들이에요:
  
  - [액션 아이템 1] (기한: [날짜])
  - [액션 아이템 2] (기한: [날짜])
  
  궁금한 점 있으시면 말씀해주세요!"
```

---

## 4단계: 주간 팔로업 자동화

```
> 이번 주 회의록들(meeting-notes-*.md)을 모두 읽고
  완료되지 않은 액션 아이템을 정리해줘.
  
  기한이 지난 것은 ⚠️ 표시.
  이번 주 안에 해야 할 것은 🔥 표시.
  
  weekly-followup-[날짜].md로 저장하고
  Slack #general에 공유해줘.
```

---

## 5단계: 회의 요약 템플릿 커스터마이징

부서나 회의 유형에 따라 다른 템플릿이 필요할 수 있어요:

```
> 우리 팀에 맞는 회의록 템플릿을 만들어줘.
  
  우리 팀 특성:
  - 부서: [부서명]
  - 주요 회의 유형: 주간 스탠드업, 프로젝트 킥오프, 리뷰 회의
  
  각 회의 유형별 맞춤 템플릿 3개.
  templates/meeting/ 폴더에 저장.
```

---

## 실전 활용: 영어 회의 통역 + 정리

글로벌 팀 회의도 자동화할 수 있어요:

```
> 다음 영어 회의록을 한국어로 번역하고 정리해줘:
  
  [영어 회의 내용]
  
  번역은 자연스러운 한국어로,
  기술 용어는 원어 병기 (예: 스프린트(sprint)).
  동일한 정리 형식 적용.
```

<Callout type="info">
회의 직후 5분 안에 Claude Code에 메모를 붙여넣는 습관을 들이면, 다음 날 회의록 정리에 시간을 쓸 필요가 없어요.
</Callout>
