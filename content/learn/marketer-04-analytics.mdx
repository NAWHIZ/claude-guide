# 마케팅 성과, 숫자로 말하기

"이 캠페인이 효과 있었나요?" — 이 질문에 즉시 답하려면 데이터가 있어야 해요. Google Analytics 숫자를 매번 캡처해서 PPT에 붙여넣는 작업, 이제 자동화할 수 있어요.

---

## 1단계: 마케팅 데이터 수집 구조 만들기

```
> 마케팅 성과 추적을 위한 Supabase 데이터베이스를 설계해줘.
  
  테이블:
  1. campaigns: 캠페인명, 채널, 예산, 시작일, 종료일
  2. daily_metrics: 날짜, 캠페인id, 노출수, 클릭수, 전환수, 비용
  3. utm_tracking: utm_source, utm_medium, utm_campaign, 방문수, 전환수
  
  SQL 생성 쿼리와 RLS 정책 포함해서 만들어줘.
```

---

## 2단계: CSV 데이터 자동 분석

Google Analytics나 광고 플랫폼에서 내보낸 CSV를 분석해요:

```
> 다음 파일들을 읽고 마케팅 성과를 분석해줘:
  - ga-export.csv (Google Analytics 데이터)
  - meta-ads.csv (메타 광고 데이터)
  - google-ads.csv (구글 광고 데이터)
  
  분석 항목:
  1. 채널별 총 비용, 클릭수, 전환수, CPA
  2. 가장 ROI 높은 캠페인 Top 3
  3. 예산 낭비가 큰 캠페인
  4. 전환율이 가장 높은 랜딩 페이지
  5. 시간대별/요일별 성과 패턴
  
  analysis-[날짜].md로 저장.
```

---

## 3단계: 자동 보고서 대시보드 만들기

```
> Next.js로 마케팅 성과 대시보드를 만들어줘.
  
  Supabase에서 데이터 조회.
  
  페이지 구성:
  1. 월간 KPI 요약 (비용, 클릭, 전환, ROAS)
  2. 채널별 성과 비교 (Bar chart)
  3. 일별 전환 트렌드 (Line chart)
  4. 캠페인별 ROI 테이블 (정렬 가능)
  5. UTM 소스별 전환 분포 (Pie chart)
  
  인증: DASHBOARD_PASSWORD 환경변수
  배포: Vercel
```

---

## 4단계: 월간 마케팅 리포트 자동 생성

```
> Supabase에서 지난 달 데이터를 읽고
  월간 마케팅 리포트를 만들어줘.
  
  포함 항목:
  1. 이번 달 성과 요약 (전월 대비)
  2. 예산 집행 현황 (계획 vs 실적)
  3. 채널별 성과 분석
  4. 잘 된 것과 개선 필요한 것
  5. 다음 달 예산 배분 추천
  
  형식: 경영진 보고용 PDF
  → HTML로 만들고 Chrome으로 PDF 변환
  
  monthly-report-[월].pdf로 저장.
```

---

## 5단계: A/B 테스트 결과 분석

```
> A/B 테스트 결과를 분석해줘.
  
  파일: ab-test-results.csv
  컬럼: variant, impressions, clicks, conversions
  
  분석:
  1. 통계적 유의성 검정 (p-value 계산)
  2. 각 variant의 전환율 + 신뢰구간
  3. 승자 variant와 그 이유
  4. 예상 연간 수익 개선 효과
  5. 다음 테스트 추천 요소
```

<Callout type="info">
A/B 테스트 결과를 분석할 때 p-value < 0.05이면 통계적으로 유의미한 차이예요. Claude Code가 이 계산을 대신해줘서 따로 통계를 공부하지 않아도 됩니다.
</Callout>

---

## 실전 팁: 마케팅 예산 최적화

```
> 지난 3개월 채널별 성과 데이터를 분석하고
  다음 달 예산 배분을 최적화해줘.
  
  총 예산: [금액]원
  
  최적화 기준:
  - CPA(고객획득비용) 최소화
  - ROAS(광고수익률) 최대화
  - 브랜드 인지도 채널 최소 20% 유지
  
  현재 배분 vs 추천 배분 비교 테이블로 보여줘.
  예상 성과 개선 효과도 계산해줘.
```
