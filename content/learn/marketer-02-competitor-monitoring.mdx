# 경쟁사 동향, 자동으로 받아보기

마케터라면 경쟁사 캠페인이 언제 뜨는지, 어떤 메시지로 광고하는지 항상 파악해야 해요. 지금까지는 직접 경쟁사 사이트를 돌아다니며 확인했다면, 이제 Claude Code가 자동으로 모아서 알려줘요.

---

## 무엇을 모니터링할 수 있나요?

| 모니터링 항목 | 방법 |
|-------------|------|
| 신규 블로그 포스트 | Tavily 검색 |
| 가격 변동 | Tavily 검색 |
| 신규 광고 캠페인 | Tavily 검색 |
| SNS 최신 포스팅 | Tavily 검색 |
| 언론 보도 | Tavily 검색 |
| 채용 동향 (전략 힌트) | Tavily 검색 |

---

## 1단계: 경쟁사 모니터링 설정 파일

```
> 마케터용 경쟁사 모니터링 설정 파일을 만들어줘.
  
  파일: monitoring-config.json
  
  포함 정보:
  {
    "competitors": [
      {
        "name": "경쟁사A",
        "website": "경쟁사A.com",
        "search_keywords": ["경쟁사A 캠페인", "경쟁사A 신기능", "경쟁사A 가격"],
        "social": "@경쟁사A"
      }
    ],
    "alert_keywords": ["가격 인하", "신규 출시", "무료", "프로모션"],
    "report_day": "월요일",
    "slack_channel": "#marketing-intel"
  }
```

---

## 2단계: 즉시 경쟁사 마케팅 분석

```
> monitoring-config.json을 읽고
  Tavily로 각 경쟁사의 최근 마케팅 동향을 조사해줘.
  
  각 경쟁사별:
  1. 최근 2주 주요 콘텐츠/캠페인
  2. 현재 강조하는 메시지 (Value Proposition)
  3. 가격 정책 변화
  4. SNS 최근 포스팅 주제
  5. 우리에게 위협이 되는 요소
  6. 우리가 대응해야 할 사항
  
  competitor-intel-[날짜].md로 저장.
```

---

## 3단계: 마케팅 메시지 비교 분석

```
> 경쟁사들의 마케팅 메시지를 비교 분석해줘.
  
  Tavily로 각 경쟁사 홈페이지 헤드라인과 광고 카피를 수집해서:
  
  1. 각 경쟁사의 핵심 메시지 1줄 요약
  2. 어떤 감정/욕구에 소구하는지
  3. 우리 메시지와 겹치는 포인트
  4. 우리가 차별화할 수 있는 영역
  5. 우리 메시지 개선 제안
  
  message-comparison.md로 저장.
```

---

## 4단계: 주간 자동 브리핑 설정

```
> 매주 월요일 아침에 자동으로 실행되는
  경쟁사 마케팅 브리핑 스크립트를 만들어줘.
  
  scripts/weekly-marketing-brief.sh
  
  실행 내용:
  1. monitoring-config.json 읽기
  2. Tavily로 각 경쟁사 7일치 동향 수집
  3. Claude로 분석 및 요약
  4. weekly-brief-[날짜].md 저장
  5. Slack #marketing-intel에 요약 전송
  
  실행: 매주 월요일 오전 8시 (cron 설정 포함)
```

---

## 5단계: 경쟁 캠페인 데이터베이스 구축

```
> Supabase에 경쟁사 캠페인 데이터베이스를 만들어줘.
  
  테이블: competitor_campaigns
  컬럼: id, competitor, date, type, headline, message, channel, notes
  
  새 캠페인 발견 시 자동으로 저장.
  6개월치 데이터가 쌓이면 트렌드 분석 가능.
```

---

## 실전 활용: 반격 캠페인 아이디어

경쟁사 동향을 파악했으면 대응 전략을 세워요:

```
> competitor-intel-[날짜].md를 읽고
  우리 브랜드 관점에서 대응 마케팅 전략을 제안해줘.
  
  brand-voice.md의 브랜드 가이드라인 적용.
  
  제안 항목:
  1. 즉시 대응 필요한 경쟁사 움직임
  2. 우리가 강조해야 할 차별점
  3. 이번 주 집중할 메시지
  4. 대응 콘텐츠 3개 아이디어 (채널별)
```

<Callout type="info">
경쟁사 모니터링은 방어용만이 아니에요. 경쟁사가 놓치고 있는 고객의 불만이나 니즈를 파악해서 선제적으로 대응하는 데도 활용할 수 있어요.
</Callout>
