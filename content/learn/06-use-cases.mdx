# 실전 활용 사례

실제로 비개발자들이 Claude Code를 어떤 식으로 활용하는지 업무 유형별로 살펴볼게요.

## 문서 및 콘텐츠 작업

### 대량 문서 처리

```
> contracts 폴더 안의 모든 계약서 파일에서 "갑"을 "발주처"로 바꿔줘
```

```
> 이 폴더의 모든 .docx 파일 목록과 각 파일의 수정 날짜를 보여줘
```

### 콘텐츠 정리 및 요약

```
> meeting_notes 폴더의 지난 달 회의록을 모두 읽고
  핵심 결정 사항만 모아서 monthly_summary.md로 만들어줘
```

```
> 이 긴 보고서를 3페이지 분량의 요약본으로 만들어줘
```

### 파일 형식 변환 준비

```
> 이 CSV 파일을 보기 좋게 정리하고, 헤더 행을 한국어로 바꿔줘
```

## 데이터 파일 작업

### CSV/데이터 분석

```
> sales.csv 파일을 읽고 월별 총 매출을 계산해줘
```

```
> customers.csv에서 서울 지역 고객만 필터링해서 seoul_customers.csv로 저장해줘
```

```
> 이 두 CSV 파일에서 중복되는 이메일 주소를 찾아줘
```

<Callout type="tip">
Claude Code는 CSV 파일을 직접 읽고 분석할 수 있어요. 엑셀 없이도 데이터를 처리할 수 있습니다.
</Callout>

### 파일 정리

```
> downloads 폴더에서 2023년보다 오래된 파일 목록을 보여줘
```

```
> 중복된 파일 이름이 있는지 찾아줘
```

## 웹사이트 수정 (HTML)

코딩을 전혀 몰라도 Claude Code로 웹사이트 파일을 수정할 수 있어요.

```
> index.html 파일에서 메뉴에 "서비스 소개" 항목을 추가해줘
```

```
> 웹사이트 모든 HTML 파일에서 푸터의 저작권 연도를 2024로 바꿔줘
```

```
> contact.html의 전화번호와 이메일 주소를 새로운 것으로 바꿔줘:
  전화: 02-1234-5678, 이메일: new@company.com
```

<BeforeAfter>
  <Before>
    웹 에이전시에 연락해서 수정 요청 → 며칠 후 변경 완료 → 비용 발생
  </Before>
  <After>
    Claude Code에게 직접 요청 → 5분 이내 변경 완료 → 무료
  </After>
</BeforeAfter>

## 업무 자동화

### 반복 작업

```
> photos 폴더 안의 모든 이미지 파일 이름 앞에 날짜 "20240326_"을 붙여줘
```

```
> 이 폴더에 있는 모든 .txt 파일을 하나의 파일로 합쳐줘.
  각 파일 이름을 구분선으로 넣어줘.
```

### 보고서 생성

```
> projects 폴더를 읽고 각 프로젝트의 상태를 정리한 주간 보고서를 만들어줘
```

## 개발자와 협업할 때

개발자가 만든 코드를 이해해야 할 때도 Claude Code가 도움이 돼요.

```
> 개발자가 보내준 이 코드 파일이 어떤 역할을 하는지 쉽게 설명해줘
```

```
> 이 설정 파일에서 서버 주소 부분이 어디인지 찾아줘
```

```
> 개발팀에서 준 이 에러 메시지가 무슨 뜻인지 설명해줘
```

## 학습과 탐구

```
> 이 폴더의 코드를 읽고 이 프로젝트가 어떻게 동작하는지 설명해줘
```

```
> package.json 파일을 보고 이 프로젝트에 어떤 기술이 사용됐는지 알려줘
```

## 실제 업무에 바로 적용하는 팁

1. **작게 시작하세요**: 처음엔 한두 개 파일로 테스트해보세요
2. **구체적으로 요청하세요**: "수정해줘" 보다 "A를 B로 바꿔줘"가 더 정확해요
3. **확인하세요**: 변경 전에 Claude가 보여주는 내용을 꼭 검토하세요
4. **반복 활용하세요**: 같은 패턴의 작업은 요청 문구를 저장해두면 편해요
