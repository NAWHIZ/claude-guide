# 실전 활용 사례 모음

Claude를 어떻게 쓰면 좋을지 감이 잘 안 잡힌다면, 실제 활용 사례를 보는 게 가장 빠른 방법이에요. 직장인, 학생, 창업자, 콘텐츠 제작자 등 다양한 상황에서 바로 써먹을 수 있는 예시를 모아봤어요.

## 📝 글쓰기 & 문서 작업

### 이메일/공문 작성
이메일을 쓸 때마다 어떻게 시작할지, 어떤 말투를 써야 할지 고민되지 않으세요? Claude에게 상황을 설명하고 대신 써달라고 해보세요.

<Callout type="tip">이렇게 물어보세요: "거래처 담당자에게 납기일을 2주 연장 요청하는 정중한 이메일을 써줘. 이유는 원자재 수급 문제고, 불편을 드려서 죄송하다는 내용도 포함해줘."</Callout>

### 보고서 초안 잡기
빈 화면 앞에서 막막할 때 Claude에게 초안을 맡겨보세요. 완성본이 아니어도 초안이 있으면 작업 속도가 훨씬 빨라져요.

<Callout type="tip">이렇게 물어보세요: "신규 고객 서비스 도입 검토 보고서의 목차를 잡아줘. 경영진 보고용이고 배경, 현황 분석, 제안 내용, 기대 효과, 예산 추정이 들어가야 해."</Callout>

### SNS 게시물 작성
인스타그램, 링크드인, 블로그 등 플랫폼마다 어울리는 글쓰기 스타일이 달라요. 플랫폼과 목적을 알려주면 딱 맞는 글을 써줘요.

<Callout type="tip">이렇게 물어보세요: "우리 카페 새 메뉴 출시를 알리는 인스타그램 캡션을 써줘. 감성적이고 따뜻한 톤으로, 해시태그 5개도 포함해줘."</Callout>

## 🔍 리서치 & 요약

### 긴 문서/논문 요약
읽기 힘든 긴 문서를 Claude에 붙여넣고 핵심만 뽑아달라고 해보세요. 계약서, 보도자료, 논문 등 어떤 형태든 OK예요.

<Callout type="tip">이렇게 물어보세요: "아래 계약서를 읽고, 내가 특히 주의해야 할 조항을 5가지로 요약해줘. 법률 용어는 쉬운 말로 바꿔서 설명해줘." [계약서 내용 붙여넣기]</Callout>

### 복잡한 개념 쉽게 설명 요청
이해하기 어려운 개념, 기술 용어, 경제 지표 등을 쉬운 말로 설명해달라고 해보세요.

<Callout type="tip">이렇게 물어보세요: "양적완화가 뭔지 금융을 전혀 모르는 사람에게 설명하듯이 알려줘. 일상적인 비유를 사용해서."</Callout>

### 비교 분석표 만들기
여러 선택지를 비교해야 할 때 표로 정리해달라고 하면 의사결정이 훨씬 쉬워져요.

<Callout type="tip">이렇게 물어보세요: "Notion, Obsidian, Roam Research를 기능, 가격, 사용 편의성, 협업 기능 기준으로 비교하는 표를 만들어줘."</Callout>

## 💼 업무 생산성

### 회의록 정리
회의 내용을 타이핑하거나 녹취록을 붙여넣고 깔끔하게 정리해달라고 해보세요.

<Callout type="tip">이렇게 물어보세요: "아래 회의 내용을 안건별로 정리하고, 결정된 사항과 담당자/기한이 있는 액션 아이템을 별도로 표로 정리해줘." [회의 내용 붙여넣기]</Callout>

### 프레젠테이션 구조 잡기
발표 자료를 만들기 전에 전체 흐름을 먼저 잡아달라고 해보세요. 구조가 잡히면 슬라이드 작성이 훨씬 빨라져요.

<Callout type="tip">이렇게 물어보세요: "신입 직원 대상 사내 보안 교육 PPT의 슬라이드 구성안을 잡아줘. 총 15분 발표 분량이고, 실습 예제도 포함되어야 해."</Callout>

### 데이터 해석 도움 요청
숫자나 통계 데이터를 어떻게 해석해야 할지 막막할 때 Claude에게 물어보세요.

<Callout type="tip">이렇게 물어보세요: "우리 앱 지난 달 지표야. DAU 1200명, 리텐션율 35%, 평균 세션 시간 4분 20초. 이 수치가 의미하는 바를 해석하고, 개선이 필요한 부분을 알려줘."</Callout>

## 🎓 학습 도우미

### 어려운 개념 쉬운 설명
공부하다 모르는 게 생기면 바로 Claude에게 물어보세요. 교과서보다 훨씬 친절하게 설명해줄 거예요.

<Callout type="tip">이렇게 물어보세요: "미분이 뭔지 수학을 싫어하는 고등학생도 이해할 수 있게 설명해줘. 실생활 예시를 들어서."</Callout>

### 외국어 번역 & 교정
영어 이메일이나 문서를 쓸 때, 또는 외국어 자료를 이해할 때 도움을 받아보세요.

<Callout type="tip">이렇게 물어보세요: "아래 영어 이메일 초안을 교정해줘. 문법 오류를 수정하고, 너무 딱딱한 표현은 더 자연스럽게 바꿔줘. 수정한 부분은 표시해줘."</Callout>

### 면접 질문 연습
취업 준비나 이직 준비 중이라면 면접 연습을 Claude와 해볼 수 있어요.

<Callout type="tip">이렇게 물어보세요: "마케팅 매니저 면접을 준비 중이야. 자주 나오는 예상 질문 10개를 만들어주고, 내가 답변하면 피드백을 줘."</Callout>

## 🚀 아이디어 & 기획

### 브레인스토밍
혼자 생각하면 맴도는 아이디어도 Claude와 함께 하면 다양한 방향으로 확장돼요.

<Callout type="tip">이렇게 물어보세요: "직장인의 점심 시간을 더 알차게 만드는 앱 아이디어를 15개 브레인스토밍해줘. 엉뚱하고 창의적인 것도 환영해."</Callout>

### 사업/프로젝트 기획
아이디어 단계에서 기획의 뼈대를 잡을 때 Claude가 든든한 파트너가 되어줘요.

<Callout type="tip">이렇게 물어보세요: "1인 유튜브 채널 시작을 기획 중이야. 채널 콘셉트 정하기부터 첫 영상 업로드까지 3개월 로드맵을 만들어줘. 부업 수준으로 주 1회 업로드 목표야."</Callout>

### 제목/슬로건 아이디어
좋은 제목이나 슬로건이 떠오르지 않을 때, 여러 버전을 빠르게 뽑아볼 수 있어요.

<Callout type="tip">이렇게 물어보세요: "친환경 텀블러 브랜드의 슬로건 아이디어 10개를 줘. 20-30대를 타겟으로, 환경 의식과 라이프스타일 감성을 동시에 담아야 해."</Callout>
