# Claude 모델 비교

Claude는 하나의 모델만 있는 게 아니에요. 용도와 성능에 따라 여러 모델이 있고, 상황에 맞는 모델을 선택하면 더 빠르고 효율적으로 작업할 수 있어요.

## 모델 이름 읽는 법

Claude 모델 이름은 이렇게 구성돼요: **Claude [세대] [크기]**

- **세대**: 3, 3.5, 3.7 등 — 숫자가 높을수록 최신 버전
- **크기/등급**: Haiku (소형) → Sonnet (중형) → Opus (대형)

예를 들어 "Claude 3.5 Sonnet"은 3.5 세대의 중간 등급 모델이에요.

## 모델별 특징 비교

| 모델 | 속도 | 비용 | 성능 | 추천 용도 |
|------|------|------|------|-----------|
| **Claude Haiku** | 매우 빠름 | 가장 저렴 | 기본~중간 | 간단한 질문, 빠른 응답, 대량 처리 |
| **Claude Sonnet** | 보통 | 중간 | 높음 | 대부분의 업무, 글쓰기, 분석 |
| **Claude Opus** | 느림 | 가장 비쌈 | 최고 | 복잡한 추론, 심층 분석, 창작 |

## 각 모델은 언제 쓰면 좋나요?

### Claude Haiku — 빠르고 가벼운 작업에

Haiku는 세 모델 중 가장 빠르게 응답해요. 복잡한 추론이 필요 없는 간단한 작업에 딱이에요.

**추천 상황:**
- 간단한 질문에 빠른 답이 필요할 때
- 짧은 텍스트 번역이나 요약
- 반복적인 작업을 많이 처리해야 할 때 (API 사용 시 비용 절감)
- 챗봇처럼 즉각적인 응답이 중요한 경우

### Claude Sonnet — 대부분의 작업에

Sonnet은 속도와 성능의 균형이 잘 잡힌 모델이에요. 일반적인 업무 대부분에 가장 적합해요. Claude.ai에서 기본으로 제공되는 모델이기도 해요.

**추천 상황:**
- 이메일, 보고서, 블로그 글 작성
- 문서 요약 및 분석
- 코드 작성 및 설명
- 아이디어 브레인스토밍
- 일상적인 업무 전반

**실제로 가장 많이 쓰이는 모델이에요.** 뭘 써야 할지 모르겠다면 Sonnet으로 시작하세요.

### Claude Opus — 복잡하고 깊이 있는 작업에

Opus는 Claude 라인업 중 가장 강력한 모델이에요. 속도가 느리고 비용도 높지만, 복잡한 문제에서는 확연히 다른 수준의 답을 내줘요.

**추천 상황:**
- 복잡한 논리 추론이나 수학 문제
- 심층 분석 및 연구
- 고품질 창작 작업 (소설, 시나리오, 고급 카피라이팅)
- 전략적 의사결정을 위한 다각적 분석
- Extended Thinking 기능 사용 시

## Claude 3.5와 3.7은 뭐가 다른가요?

같은 Sonnet이라도 세대가 다르면 성능이 달라요.

- **Claude 3 Sonnet**: 초기 Claude 3 계열
- **Claude 3.5 Sonnet**: 코딩 능력과 명령 이해 능력이 크게 향상된 버전. 출시 당시 많은 벤치마크에서 경쟁 모델을 앞섰어요.
- **Claude 3.7 Sonnet**: Extended Thinking 기능 지원, 복잡한 추론 성능이 더욱 향상된 최신 버전

일반적으로 숫자가 높은 최신 세대 모델이 더 좋은 성능을 보여요. Claude.ai에서는 보통 최신 모델이 기본값으로 설정돼 있어요.

## Claude.ai에서 모델 선택하는 방법

1. 대화 입력창 위쪽 또는 화면 상단에서 현재 선택된 모델 이름을 찾으세요.
2. 모델 이름을 클릭하면 사용 가능한 모델 목록이 펼쳐져요.
3. 원하는 모델을 선택하면 다음 대화부터 해당 모델로 전환돼요.

## 어떤 모델을 선택해야 할지 모르겠다면?

간단한 기준을 드릴게요:

- **빠른 답변이 우선**: Haiku
- **일반적인 업무**: Sonnet (이게 정답인 경우가 90%)
- **중요한 분석이나 복잡한 작업**: Opus
- **모르겠으면**: Sonnet으로 시작하고, 결과가 아쉬우면 Opus로 올려보세요
