## 시작하기 전에

**Q. 코딩을 전혀 모르는데 Claude Code를 쓸 수 있나요?**

네, 충분히 사용할 수 있어요. Claude Code는 자연어(한국어)로 대화하면서 사용하는 도구예요. 코딩 지식이 없어도 파일 관리, 문서 작업, 데이터 처리 등 다양한 작업을 할 수 있어요. 오히려 코딩을 모르는 분들에게 더 유용할 수 있어요.

**Q. Claude.ai(웹사이트)와 Claude Code의 차이는 무엇인가요?**

Claude.ai는 웹브라우저에서 사용하는 채팅 서비스예요. Claude Code는 터미널에서 실행하는 도구로, 컴퓨터 파일을 직접 열고, 읽고, 수정하고, 만들 수 있어요. 파일을 실제로 다룰 수 있다는 점이 가장 큰 차이예요.

**Q. Claude Code는 무료인가요?**

Claude Code 자체는 무료로 설치할 수 있어요. 다만 사용할 때마다 API 요금이 발생해요. 신규 가입 시 무료 크레딧이 제공되며, 일반적인 사용량이라면 월 몇 달러 수준이에요.

## 설치 관련

**Q. Node.js가 뭔지 모르는데 꼭 설치해야 하나요?**

네, Claude Code가 Node.js 기반으로 만들어져서 반드시 필요해요. 하지만 복잡하게 생각하지 않아도 돼요. [nodejs.org](https://nodejs.org)에서 다운로드해서 설치하면 끝이에요. 설치 후에는 신경 쓰지 않아도 됩니다.

**Q. API 키를 잃어버렸어요. 어떻게 하나요?**

[console.anthropic.com](https://console.anthropic.com)에 로그인해서 새 API 키를 만들면 돼요. 기존 키는 비활성화하고 새 키를 사용하세요.

**Q. `claude: command not found` 오류가 나와요.**

터미널을 완전히 닫고 새로 열어보세요. 그래도 안 되면 [상세 설치 가이드](/reference/installation-guide)의 문제 해결 섹션을 참고하세요.

## 사용 중 궁금한 것

**Q. Claude Code가 내 개인 파일을 보는 게 안전한가요?**

Claude Code는 여러분이 실행한 폴더에 있는 파일에만 접근해요. 또한 파일을 수정하기 전에 항상 확인을 요청해요. 단, API를 통해 Anthropic 서버로 텍스트가 전송되므로, 아주 민감한 기밀 정보가 담긴 파일은 주의가 필요해요.

**Q. Claude Code가 파일을 잘못 수정했어요. 되돌릴 수 있나요?**

Claude Code에 "방금 변경한 파일을 원래대로 되돌려줘"라고 요청하면 돼요. 또는 git을 사용하는 프로젝트라면 `git checkout`으로 복구할 수 있어요. 중요한 파일은 작업 전에 백업해두는 습관이 좋아요.

**Q. 한 번에 얼마나 많은 파일을 처리할 수 있나요?**

기술적 제한은 없지만, 파일이 많을수록 처리 시간이 길어지고 API 비용이 증가해요. 수십 개 파일까지는 문제없이 처리해요.

**Q. 대화 기록은 얼마나 기억하나요?**

현재 세션의 대화는 기억하지만, Claude Code를 종료하면 대화 기록은 사라져요. CLAUDE.md 파일에 중요한 정보를 저장하면 다음에 실행할 때도 맥락을 유지할 수 있어요.

**Q. 인터넷에서 정보를 검색해줄 수 있나요?**

기본적으로는 인터넷 검색 기능이 없어요. 로컬 파일 작업에 특화되어 있어요. 웹 검색이 필요하다면 Claude.ai 웹사이트를 이용하는 게 좋아요.

## 요금 관련

**Q. 예상치 못한 큰 요금이 청구될까봐 걱정돼요.**

[console.anthropic.com](https://console.anthropic.com)에서 월 최대 사용 한도를 설정할 수 있어요. 한도에 도달하면 자동으로 사용이 중단돼요. 또한 대화 내역에서 사용량을 실시간으로 확인할 수 있어요.

**Q. Claude.ai Pro 구독이 있으면 Claude Code도 무료인가요?**

아니에요. Claude.ai Pro 구독과 Claude Code API 요금은 완전히 별개예요. Claude Code 사용 시에는 API 요금이 별도로 청구돼요.
