## Claude Code 요금 구조

Claude Code는 **사용한 만큼만 내는 방식**이에요. 구독료가 없고, API 사용량에 따라 요금이 청구됩니다.

> **참고**: 정확한 최신 요금은 [anthropic.com/pricing](https://www.anthropic.com/pricing)에서 확인하세요. 이 페이지의 요금은 변동될 수 있어요.

## 토큰이란?

요금은 "토큰" 단위로 계산돼요. 토큰은 텍스트를 처리하는 단위로, 대략:
- 영어 단어 1개 ≈ 1~2 토큰
- 한국어 글자 1개 ≈ 2~3 토큰
- 코드 한 줄 ≈ 5~20 토큰

## 모델별 요금 비교

| 모델 | 입력 (1M 토큰) | 출력 (1M 토큰) | 특징 |
|------|---------------|---------------|------|
| Claude Opus 4 | $15 | $75 | 가장 강력 |
| Claude Sonnet 4 | $3 | $15 | 균형 잡힌 성능 (권장) |
| Claude Haiku 3.5 | $0.80 | $4 | 빠르고 저렴 |

## 실제 사용 예시

일반적인 작업별 예상 비용 (Sonnet 기준):

| 작업 | 예상 토큰 | 예상 비용 |
|------|-----------|----------|
| 파일 하나 읽고 요약 | ~2,000 토큰 | $0.01 미만 |
| 10개 파일 일괄 수정 | ~10,000 토큰 | ~$0.05 |
| 복잡한 분석 보고서 작성 | ~30,000 토큰 | ~$0.15 |
| 대규모 코드 리팩토링 | ~100,000 토큰 | ~$0.50 |

하루에 여러 시간 적극적으로 사용해도 월 $10~30 수준이에요.

## 무료로 시작하기

Anthropic은 신규 가입자에게 **무료 크레딧**을 제공해요. 이 크레딧으로 Claude Code를 충분히 체험해볼 수 있어요.

[console.anthropic.com](https://console.anthropic.com)에서 가입하면 자동으로 적용됩니다.

## 비용 절약 팁

### 적절한 모델 선택

- 간단한 파일 읽기/수정 → **Haiku** 사용
- 일반 업무 → **Sonnet** 사용 (기본값 권장)
- 복잡한 분석 → **Opus** 사용

### 불필요한 컨텍스트 줄이기

- 관련 없는 대화가 길어지면 `/clear`로 초기화
- 큰 파일 전체 읽기 대신 필요한 부분만 요청

### 사용량 모니터링

[console.anthropic.com](https://console.anthropic.com)에서 실시간 사용량과 비용을 확인할 수 있어요. 예산 한도(Budget Limit)도 설정 가능해요.

## 결제 방법

- 신용카드 / 직불카드
- 후불 방식 (사용 후 월별 청구)
- 선불 크레딧 충전도 가능

## 자주 묻는 요금 질문

**Q. 구독 없이 사용할 수 있나요?**

네. Claude Code는 구독료 없이 사용량만큼만 내는 종량제예요.

**Q. Claude.ai 구독(Pro 플랜)과 API 요금은 별개인가요?**

네. Claude.ai 웹사이트 구독과 API(Claude Code) 요금은 완전히 별개예요. Claude Code는 API 요금만 사용해요.

**Q. 예산 한도를 설정할 수 있나요?**

네. Anthropic 콘솔에서 월 최대 지출 한도를 설정할 수 있어요. 한도에 도달하면 자동으로 사용이 중단돼요.
