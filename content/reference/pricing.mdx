# 요금제 안내

Claude를 쓰다 보면 "유료로 전환해야 하나?" 하는 생각이 들 때가 있어요. 어떤 플랜이 있는지, 무엇이 다른지 정리해드릴게요.

## 플랜 한눈에 비교

| 항목 | Free | Pro | Team |
|------|------|-----|------|
| **가격** | 무료 | $20/월 | $30/월/인 |
| **사용량** | 제한적 | 훨씬 많음 | Pro보다 많음 |
| **모델 선택** | 기본 모델 | 전체 모델 | 전체 모델 |
| **Projects** | 제한적 | 사용 가능 | 사용 가능 |
| **파일 업로드** | 제한적 | 사용 가능 | 사용 가능 |
| **컨텍스트 길이** | 기본 | 길음 | 더 길음 |
| **팀 공유** | 불가 | 불가 | 가능 |
| **관리자 기능** | 없음 | 없음 | 있음 |

## 각 플랜 자세히 보기

### Free 플랜 — 무료로 시작하기

무료 플랜은 처음 Claude를 경험해보는 데 충분해요. 하루에 정해진 횟수만큼 대화를 나눌 수 있고, 기본적인 기능은 모두 사용할 수 있어요.

**이런 분께 추천해요:**
- Claude를 처음 써보는 분
- 가끔씩만 사용하는 분
- 기능을 먼저 체험해보고 싶은 분

**한계점:**
- 하루 사용량 한도가 있어요. 한도에 도달하면 다음 날까지 기다려야 해요.
- 접속이 많은 시간대에는 속도가 느려질 수 있어요.
- 최신 고성능 모델을 선택하지 못할 수 있어요.

### Pro 플랜 — 매일 쓰는 분을 위한 선택

월 $20(약 27,000원)의 Pro 플랜은 Claude를 본격적으로 업무에 활용하는 분들에게 가장 인기 있는 선택이에요.

**Pro에서 달라지는 것들:**
- 훨씬 많은 사용량 (Free의 5배 이상)
- Haiku, Sonnet, Opus 모든 모델 선택 가능
- Projects 기능 완전 사용 (대화 간 맥락 유지)
- 파일 및 문서 업로드 가능 (PDF, Word 등)
- 더 긴 컨텍스트 윈도우
- Extended Thinking 기능 사용 가능

**이런 분께 추천해요:**
- 매일 업무에 Claude를 사용하는 분
- 파일을 업로드해서 분석하고 싶은 분
- 프로젝트 기능으로 반복 작업을 효율화하고 싶은 분
- 최신 고성능 모델을 쓰고 싶은 분

### Team 플랜 — 팀과 함께 쓰는 선택

월 $30/인의 Team 플랜은 여러 명이 함께 Claude를 활용하는 조직을 위한 플랜이에요. 최소 2인부터 사용 가능해요.

**Team에서 추가되는 것들:**
- Pro의 모든 기능 포함
- 팀원 간 Projects 공유 가능
- 더 긴 컨텍스트 윈도우 (200K 토큰)
- 관리자 대시보드 (팀원 관리, 사용 현황 확인)
- 팀 전용 데이터 처리 정책

**이런 분께 추천해요:**
- 팀 단위로 Claude를 도입하려는 경우
- 공통 작업 맥락을 팀원들과 공유하고 싶은 경우
- IT 관리자가 팀 사용을 관리해야 하는 경우

## API 요금 — 개발자를 위한 사용 방식

API는 프로그램에서 직접 Claude를 호출할 때 쓰는 방식이에요. 고정 월정액이 아닌 **사용한 만큼 지불하는** 방식이에요.

**토큰 기반 요금이란?**

AI는 텍스트를 "토큰" 단위로 처리해요. 한 단어는 보통 1~2 토큰 정도예요. "안녕하세요"는 약 3~4 토큰이에요.

API 요금은 입력 토큰(내가 보낸 내용)과 출력 토큰(Claude가 답한 내용)을 합산해서 청구돼요. 모델마다 토큰당 가격이 다르고, 고성능 모델일수록 비싸요.

개발자가 아니라면 API 요금은 크게 신경 쓰지 않아도 돼요. Claude.ai 월정액 플랜으로 충분히 사용할 수 있어요.

## 나에게 맞는 플랜 고르기

**가끔씩, 궁금할 때만 쓴다 → Free**
충분히 경험해볼 수 있어요. 사용량 한도에 자주 걸린다면 Pro를 고민해보세요.

**매일 업무에 활용한다 → Pro**
하루에 몇 번 이상 Claude를 쓰거나, 파일 업로드나 프로젝트 기능이 필요하다면 Pro가 훨씬 편해요. 한 달에 한 번만 대화 한도에 걸려도 Pro가 더 나을 수 있어요.

**팀과 함께 쓴다 → Team**
3명 이상이 함께 쓴다면 Team 플랜이 효율적이에요. 공통 프로젝트 공유와 팀 관리 기능이 추가돼요.

<Callout type="tip">Pro 플랜은 경우에 따라 첫 달 무료 체험이 제공되기도 해요. claude.ai 결제 페이지에서 현재 적용 가능한 프로모션을 확인해보세요. 결제 전 언제든지 취소할 수 있어요.</Callout>

요금제는 언제든지 변경하거나 취소할 수 있어요. 처음엔 무료로 시작해서, 한도가 자주 걸린다 싶으면 Pro로 올리는 게 좋은 방법이에요.
