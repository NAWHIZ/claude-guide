# 용어 사전

Claude를 쓰다 보면 낯선 단어들을 만나게 돼요. 어렵게 느껴지는 AI 용어들을 최대한 쉽게 풀어 설명했어요.

---

## 기본 용어

**프롬프트 (Prompt)**
Claude에게 보내는 질문이나 지시 내용이에요. "프롬프트를 잘 쓴다"는 것은 Claude가 원하는 답을 잘 낼 수 있게 질문하는 방법을 안다는 뜻이에요. 입력창에 치는 모든 내용이 프롬프트예요.

**컨텍스트 (Context)**
Claude가 현재 대화에서 기억하고 있는 내용 전체를 말해요. 같은 대화 안에서 앞에 했던 말이 모두 컨텍스트가 돼요. 대화가 길어질수록 초반 내용을 잊을 수 있어요.

**컨텍스트 윈도우 (Context Window)**
Claude가 한 번에 처리할 수 있는 텍스트의 최대량이에요. "창문"처럼 이 범위 안의 내용만 볼 수 있어요. Claude 모델마다 크기가 다르고, 큰 모델일수록 더 많은 내용을 한 번에 기억할 수 있어요.

**토큰 (Token)**
AI가 텍스트를 처리하는 단위예요. 대략 한 단어가 1~2 토큰 정도예요. "안녕하세요"는 약 3~4 토큰, 영어 단어 "hello"는 1 토큰이에요. API 사용 요금은 사용한 토큰 수로 계산해요.

**시스템 프롬프트 (System Prompt)**
대화 전체에 적용되는 기본 설정 지시문이에요. 일반 메시지와 달리 대화가 시작되기 전에 Claude의 행동 방식을 미리 설정해요. Claude.ai에서는 Projects의 Custom Instructions가 이 역할을 해요.

## AI 관련 용어

**환각 (Hallucination)**
AI가 실제로 없는 정보를 있는 것처럼 그럴듯하게 말하는 현상이에요. 없는 논문을 인용하거나, 틀린 날짜를 맞는 것처럼 말하는 경우가 여기에 해당해요. 자신 있어 보인다고 항상 맞는 건 아니에요.

**LLM (Large Language Model)**
Claude, ChatGPT 같은 대형 언어 모델의 약자예요. 방대한 텍스트 데이터를 학습해서 자연스러운 언어를 생성하는 AI 모델이에요. "대형"이라는 말은 모델의 크기(파라미터 수)가 매우 크다는 뜻이에요.

**파인튜닝 (Fine-tuning)**
이미 학습된 AI 모델을 특정 목적에 맞게 추가 학습시키는 것이에요. 예를 들어 의료 문서에만 특화되도록 추가 학습시키면 그 분야에서 더 정확한 답을 줄 수 있어요.

**프롬프트 엔지니어링 (Prompt Engineering)**
AI에게 더 나은 답변을 얻기 위해 프롬프트(질문/지시)를 설계하는 기술이에요. 어렵게 들리지만, 본질은 "상대방이 잘 이해할 수 있게 말하기"와 비슷해요.

**RAG (Retrieval-Augmented Generation)**
AI가 답변을 생성할 때 외부 데이터베이스에서 관련 정보를 검색해서 활용하는 기술이에요. AI의 지식 마감일 문제를 일부 해결할 수 있어요. 기업용 AI 솔루션에서 자주 쓰여요.

**멀티모달 (Multimodal)**
텍스트만이 아니라 이미지, 음성, 동영상 등 여러 형태의 데이터를 동시에 처리할 수 있는 AI를 말해요. Claude도 텍스트뿐 아니라 이미지를 이해하는 멀티모달 기능이 있어요.

**파라미터 (Parameter)**
AI 모델이 학습을 통해 얻은 내부 설정값이에요. "수십억 파라미터 모델"이라는 표현은 이 값이 수십억 개라는 뜻이고, 일반적으로 파라미터가 많을수록 더 복잡한 패턴을 학습할 수 있어요.

## Claude 전용 용어

**아티팩트 (Artifact)**
Claude가 생성한 문서, 코드, HTML, SVG 등의 결과물을 대화창 오른쪽 별도 패널에 표시하는 기능이에요. 대화가 길어도 결과물이 고정된 패널에 남아있어서 편리해요.

**프로젝트 (Projects)**
여러 대화에 걸쳐 맥락을 유지할 수 있는 폴더 기능이에요. 프로젝트에 저장된 지침과 파일은 그 프로젝트의 모든 대화에 자동으로 적용돼요. 반복 작업 시 매번 배경 설명을 다시 하지 않아도 돼요.

**Extended Thinking**
Claude가 답하기 전에 더 깊이, 더 오래 생각하는 모드예요. 복잡한 수학, 논리 추론, 전략 분석 같은 작업에서 훨씬 정교한 답을 얻을 수 있어요. Claude 3.7 이상에서 사용 가능해요.

**지식 마감일 (Knowledge Cutoff)**
Claude가 학습한 데이터의 시간적 한계예요. 이 날짜 이후의 정보는 모를 수 있어요. 최신 뉴스, 새 법규, 최근 출시된 제품 정보 등은 직접 제공해줘야 정확한 답을 받을 수 있어요.

**Claude.ai**
Anthropic이 운영하는 Claude 공식 웹 인터페이스 주소(claude.ai)이자 서비스 이름이에요. 별도 설치 없이 브라우저에서 바로 Claude와 대화할 수 있어요.

**Anthropic**
Claude를 만든 AI 안전 연구 회사예요. 2021년 설립되었으며, AI를 안전하고 신뢰할 수 있게 만드는 것을 핵심 목표로 삼고 있어요.

## API & 개발 용어

**API (Application Programming Interface)**
프로그램에서 Claude를 직접 사용하는 방법이에요. 개발자가 자신의 앱이나 서비스에 Claude를 내장할 때 사용해요. 일반 사용자는 API를 직접 다루지 않아도 돼요.

**API 키 (API Key)**
API를 사용할 때 필요한 인증 코드예요. 비밀번호처럼 본인만 알아야 하고, 절대 공개된 곳에 올리면 안 돼요.

**노코드 (No-code)**
프로그래밍 없이 소프트웨어를 만들거나 자동화를 구성하는 방법이에요. Zapier, Make 같은 도구를 사용하면 코딩 없이 Claude를 다른 서비스와 연동할 수 있어요.

<Callout type="tip">모르는 용어가 나오면 Claude에게 직접 물어보는 것도 좋은 방법이에요. "~~가 무슨 뜻이에요? 쉽게 설명해줘"라고 하면 바로 알려줘요.</Callout>
