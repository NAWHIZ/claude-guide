## A

**API (Application Programming Interface)**
프로그램끼리 대화하는 방법이에요. Claude Code는 API를 통해 Anthropic의 서버에 있는 Claude에 접속해요. 우리가 식당에서 메뉴를 주문하면 주방에서 요리해주는 것처럼, API는 "주문을 전달하는 웨이터" 역할이에요.

**API 키 (API Key)**
Claude Code가 Anthropic 서버에 접속하기 위한 비밀 열쇠예요. `sk-ant-api03-...`로 시작하는 긴 문자열이에요. 절대 다른 사람과 공유하면 안 돼요.

**Anthropic**
Claude를 만든 AI 안전 연구 회사예요. Claude Code도 Anthropic이 개발했어요.

## C

**CLI (Command Line Interface)**
터미널에서 텍스트로 명령을 입력하는 방식이에요. Claude Code는 CLI 도구예요.

**CLAUDE.md**
프로젝트 폴더에 만들어두는 특별한 파일이에요. Claude Code를 실행할 때마다 자동으로 읽어서 프로젝트 맥락을 기억해요. "Claude에게 주는 업무 지침서"라고 생각하면 돼요.

**컨텍스트 (Context)**
Claude가 현재 대화에서 기억하고 있는 내용 전체예요. 대화가 길어질수록 컨텍스트가 커지고, 한계에 다가가면 성능이 저하될 수 있어요. `/clear`나 `/compact`로 관리해요.

## G

**git**
파일의 변경 내역을 기록하고 관리하는 도구예요. 개발자들이 많이 사용해요. Claude Code는 git 저장소를 이해하고 도움을 줄 수 있어요.

## M

**모델 (Model)**
Claude의 AI 엔진 버전이에요. Opus, Sonnet, Haiku가 있으며 성능과 비용이 달라요. `/model` 명령어로 변경할 수 있어요.

## N

**Node.js**
JavaScript를 컴퓨터에서 실행할 수 있게 해주는 프로그램이에요. Claude Code가 Node.js 기반으로 만들어져서 Claude Code를 설치하기 전에 Node.js를 먼저 설치해야 해요.

**npm (Node Package Manager)**
Node.js 패키지(프로그램)를 설치하고 관리하는 도구예요. `npm install -g @anthropic-ai/claude-code`로 Claude Code를 설치해요.

## P

**프롬프트 (Prompt)**
Claude에게 보내는 질문이나 지시예요. 더 구체적인 프롬프트일수록 더 정확한 답변을 받을 수 있어요.

## S

**슬래시 명령어 (Slash Command)**
`/`로 시작하는 Claude Code의 특별한 명령어예요. `/help`, `/clear`, `/model` 등이 있어요.

## T

**터미널 (Terminal)**
텍스트로 컴퓨터에 명령을 내리는 프로그램이에요. Mac은 "터미널(Terminal.app)", Windows는 "PowerShell"이에요. Claude Code는 터미널에서 실행돼요.

**토큰 (Token)**
Claude가 텍스트를 처리하는 단위예요. 요금은 사용한 토큰 수에 따라 계산돼요. 영어 단어 하나가 대략 1~2 토큰, 한국어 글자 하나가 2~3 토큰 정도예요.

## W

**워크스페이스 (Workspace)**
Claude Code가 작업하는 현재 폴더예요. 터미널에서 이동한 폴더가 기본 워크스페이스가 돼요.
