## 시스템 요구사항

| 항목 | 최소 요구사항 |
|------|--------------|
| OS | macOS 12+, Windows 10+, Ubuntu 20.04+ |
| Node.js | v18 이상 (v20 LTS 권장) |
| 인터넷 | 필수 (API 통신) |
| 디스크 | 500MB 이상 여유 공간 |

## Mac 설치

### Node.js 설치

**방법 1 — 공식 설치 파일 (권장):**

1. [nodejs.org](https://nodejs.org) 접속
2. "LTS" 버튼 클릭하여 `.pkg` 파일 다운로드
3. 다운로드된 파일 실행 후 안내에 따라 설치

**방법 2 — Homebrew:**

```bash
brew install node
```

### Claude Code 설치

```bash
npm install -g @anthropic-ai/claude-code
```

### API 키 영구 설정

터미널을 열 때마다 API 키를 다시 입력하지 않으려면 쉘 설정 파일에 추가하세요.

**zsh 사용 시 (Mac 기본):**

```bash
echo 'export ANTHROPIC_API_KEY=여기에API키입력' >> ~/.zshrc
source ~/.zshrc
```

**bash 사용 시:**

```bash
echo 'export ANTHROPIC_API_KEY=여기에API키입력' >> ~/.bashrc
source ~/.bashrc
```

## Windows 설치

### Node.js 설치

1. [nodejs.org](https://nodejs.org) 접속
2. "LTS" 버튼 클릭하여 `.msi` 파일 다운로드
3. 설치 파일 실행 후 "Next"를 눌러 설치 완료
4. **PowerShell을 새로 열어주세요** (기존 창에는 적용 안 됨)

### Claude Code 설치

PowerShell에서:

```powershell
npm install -g @anthropic-ai/claude-code
```

### API 키 영구 설정

PowerShell에서:

```powershell
[System.Environment]::SetEnvironmentVariable('ANTHROPIC_API_KEY', '여기에API키입력', 'User')
```

PowerShell을 재시작하면 적용돼요.

## Linux (Ubuntu/Debian) 설치

### Node.js 설치

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Claude Code 설치

```bash
npm install -g @anthropic-ai/claude-code
```

### API 키 영구 설정

```bash
echo 'export ANTHROPIC_API_KEY=여기에API키입력' >> ~/.bashrc
source ~/.bashrc
```

## API 키 발급 방법

1. [console.anthropic.com](https://console.anthropic.com) 접속
2. 계정 없으면 "Sign up"으로 가입
3. 좌측 메뉴 **"API Keys"** 클릭
4. **"Create Key"** 버튼 클릭
5. 키 이름 입력 (예: `my-claude-code`) 후 생성
6. `sk-ant-api03-...`로 시작하는 키 복사하여 저장

> **주의**: API 키는 생성 직후만 전체 내용을 볼 수 있어요. 반드시 복사해서 안전한 곳에 보관하세요.

## 설치 확인

```bash
# Node.js 버전 확인
node --version
# v20.x.x 출력되면 OK

# Claude Code 버전 확인
claude --version
# 버전 번호 출력되면 OK

# API 키 확인 및 실행
claude
# Welcome 화면이 나오면 설치 완료!
```

## 업데이트 방법

Claude Code를 최신 버전으로 업데이트하려면:

```bash
npm update -g @anthropic-ai/claude-code
```

## 문제 해결

### `claude: command not found` 오류

npm 전역 설치 경로가 PATH에 없을 때 발생해요.

**Mac/Linux:**
```bash
export PATH="$PATH:$(npm bin -g)"
```

**Windows:** npm 전역 폴더를 환경 변수 PATH에 추가하세요.

### `EACCES permission denied` 오류

권한 문제예요. Mac/Linux에서:

```bash
sudo npm install -g @anthropic-ai/claude-code
```

### API 키 오류

- 키가 `sk-ant-api03-`로 시작하는지 확인
- 키 앞뒤에 공백이 없는지 확인
- [console.anthropic.com](https://console.anthropic.com)에서 키가 활성화 상태인지 확인
